# IFC Demo Files

This workspace contains small sample IFC files for demoing the `wolfnepomuk.vscode-ifc` extension. Please only share the link and password with trusted individuals.

The files are public demo samples. Do not upload private or sensitive models to this temporary environment.

## Usage

- Open an IFC file (.ifc, .step, or .stp) in Visual Studio Code.
- Review diagnostics directly in the editor and Problems view.
- Hover over STEP IDs such as #12345 or IFC entity names such as IFCWALL.
- Use F12 or Ctrl/Cmd-click for go to definition.
- Use Shift + F12 for find references.

For a better experience, it is recommended to use the latest version of [Visual Studio Code](https://code.visualstudio.com/) on your personal computer with the [`wolfnepomuk.vscode-ifc`](https://marketplace.visualstudio.com/items?itemName=wolfnepomuk.vscode-ifc) extension installed.
