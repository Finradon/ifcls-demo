# IFC Demo Files

This workspace contains small sample IFC files for trying the `wolfnepomuk.vscode-ifc` extension.

1. Open one of the `.ifc` files from the Explorer.
2. Use the IFC extension commands or editor features from the Command Palette.
3. Try the IFC2x3, IFC4, and IFC4x3 samples to compare schema handling.

The files are public demo samples. Do not upload private or sensitive models to this temporary environment.
